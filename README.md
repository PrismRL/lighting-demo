# lighting demo

This is the [prism template](https://github.com/PrismRL/prism-template) with the minimum required 
to implement the lighting module. The [level state](https://github.com/PrismRL/lighting-demo/blob/main/modules/game/gamestates/gamelevelstate.lua) has all of the changes documented.
